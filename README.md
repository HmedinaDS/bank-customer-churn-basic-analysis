# Bank Customer Churn Analysis

## Project Overview
This repository contains a complete data analysis project focused on predicting bank customer attrition. The project includes data cleaning, outlier treatment, and exploratory data analysis (EDA).

## Insights Discovered
* **Age:** Older customers have a higher probability of leaving the bank (strong positive correlation of 0.29).
* **Product Density Paradox:** Customers holding 3 or 4 products show a dramatic increase in churn rate compared to those with 2 products.
* **Member Activity:** Inactive members are significantly more prone to attrition.

## Tech Stack
* **Language:** R
* **Libraries:** `tidyverse`, `ggplot2`, `dplyr`, `tidyr`, `readr`
